What is Stylus?

  • CSS preprocessor used on MDN (and other Mozilla websites)
  • "Expressive, dynamic, robust"

What does it look like?

Technical requirements

Just the stylus binary!

Using Stylus in MDN

  1. Add a stylesheet to the /media/redesign/stylus directory
  2. Add an entry to MINIFY_BUNDLES (Example)
  3. Add the stylesheet to a template (Example)
  4. To test, run compile-stylesheets --watch on your host

The Good

  • Way better than just CSS (especially for maintenance)
  • Surprisingly powerful language features
  • Responsive team

The Bad

  • No source maps (Less and SASS have them)
  • Can be buggy (conditional, bidi-style, @font-face linebreaks)
